#  Modern Single Page Application (SPA) 

## Project Overview

The application is built using **Vue 3, TypeScript, and Tailwind CSS**, and it consumes data from the **DummyJSON API**. The system demonstrates modern front-end development practices including component-based architecture, API integration, and responsive UI design.


##  Objectives

* Implement a structured Vue 3 application using the Composition API
* Use TypeScript for strict type safety
* Fetch and display real-time API data
* Design a responsive user interface using Tailwind CSS
* Apply component-based architecture

---

##  Technologies Used

* Vue 3 (Composition API)
* TypeScript
* Vite
* Tailwind CSS
* DummyJSON REST API

---

##  Features Implemented

*  Fetch data from DummyJSON API (`/products`)
*  Search functionality for products
*  Category-based filtering
*  Product detail view (via route or modal)
*  Fully responsive design (mobile, tablet, desktop)



##  Installation Instructions


### 1. Open project in terminal

Navigate to the project directory:

```bash
cd your-project-folder
```

### 2. Install dependencies

```bash
npm install
```

---

##  Running the Application

```bash
npm run dev
```

Open your browser and go to:

```
http://localhost:5173
```

---

##  API Integration

This project uses the DummyJSON API:

👉 https://dummyjson.com/

Example endpoint used:

```
https://dummyjson.com/products
```
